### **_Technologies used in the project:_**

- UI: React + webpack + hot modules + D3 for Data Visualization
- Backend: Express + NodeJS + Passport + Socket.io
- ORM: Sequelize
- Database: PostgreSQL
- HTTP server: NGINX
- Deploy: Vagrant
